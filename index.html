LLM Intent Recognition

AI understands what you mean

Every voice segment is classified — content input, edit command, or send. No mode switching needed. Just speak naturally and the AI figures out the rest. You can say "change 3 to 5" using the regular voice button and the AI will detect it as an edit. The dedicated edit button uses a specialized prompt for more precise edits.

How It Works

Voice to any app, in four steps

01

Speak

Hold the record button and speak naturally. Dictate content in any language — hold-to-record (PTT) or click-to-toggle, your choice.

02

Process

Your speech-to-text model transcribes your voice. The LLM classifies intent, auto-corrects errors, applies your custom dictionary, and updates the buffer.

03

Edit (optional)

Want to refine? Just say "change 3 to 5" using the voice button — the AI detects it as an edit automatically. For more precise control, use the dedicated edit button which uses a specialized prompt focused on editing.

04

Send

Say "send" and the text goes directly into the active app — Slack, Gmail, VS Code, any text field. Native text input on every platform.

Roadmap

What's coming next

Platforms

macOS, iOS & Windows

Native apps for all major platforms. macOS menu bar, iOS standalone app, Windows system tray. Same voice-to-text workflow everywhere.

Streaming

Real-time STT

WebSocket streaming with Deepgram and Google Cloud Speech. See text appear as you speak. Optional alongside batch mode STT.

Security

Voiceprint Auth

Local voiceprint verification so only your voice can control the app. Runs on-device, no cloud biometric data.

Extensibility

Plugin System

JavaScript plugin engine with hooks at every stage — translate before sending, inject RAG context, apply templates, trigger automations.

AI

Silero VAD v6

Upgrade from v4 to v6 for improved speech detection accuracy. Pending ONNX export fix for LSTM state compatibility.

Accessibility

Whisper Speech Mode

Adaptive gain and optimized VAD sensitivity for quiet environments. Works when you whisper in a library, office, or late at night.

Productivity

Meeting Notes

Speaker diarization powered by voiceprint — automatically label who said what. LLM summarizes, extracts action items, and generates meeting minutes.

Community

Plugin Store

Community plugin directory. Share and discover plugins for translation, automation, custom STT/LLM providers, and more.

Context

Smart Mode Switching

Auto-switch modes based on the active app, scene, or GPS location. Formal tone in Gmail, concise in Slack, friendly in WhatsApp, structured in Notion, local language near travel destinations.

Personalization

Custom Modes

Create preset profiles with unique prompts, language, dictionary, and tone. Switch between business email, casual chat, technical docs, and meeting notes instantly.

Learning

Continuous AI Learning

AI summarizes your past inputs, auto-adds frequent words to your dictionary, and refines prompts to match your speaking style. All data stays local.

Social Intelligence

Reply Suggestions

AI reads the room — suggests polite phrasing for formal emails, hints at reading the air in Japanese business, and softens tone when emotions run high.

Website

Upvote or downvote features and roadmap items directly on the website. Most requested features rise to the top. IP + device fingerprint rate limiting to prevent abuse.
